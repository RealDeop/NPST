Commands

#end


Main Use Change the specfied int or str to the next value

Example Use And Outputs example: [ hi user/int 1 #end hi user/str a]

Output: hi user1 hi userb

#start


Main Use Set the prevalue for what to change to anything excpet 1 Mainly Used At Start of the script.txt file

Example Use And Outputs example: [ #start 5 hi user/int 1 #end hi user/str b]

Output: hi user5 hi userf

#set


Main Use Set the prevalue for what to change to anything

Example Use And Outputs example: [ hi user/int 1 #end hi user/int 1 #set 5 hi user/int 1]

Output: hi user1 hi user2 hi user5

#stop


Main Use Set the prevalue for what to change to anything

Example Use And Outputs example: [ hi user/int 1 #end hi user/str j #stop #set 5 hi user/int 1 ]
